# sitemap2pdf.py

A Python script to convert a sitemap to a PDF file.

## Installation

1. Clone the repository: `git clone https://github.com/username/sitemap2pdf.git`
2. Install the required packages: `pip install -r requirements.txt`

## Usage

```
python sitemap2pdf.py <sitemap_url> <output_filename> [--limit <limit>] [--hide_classes <hide_classes>] [--no_cache]
```

- `sitemap_url`: The URL of the sitemap to convert.
- `output_filename`: The name of the output PDF file.
- `--limit`: (Optional) The maximum number of URLs to convert.
- `--hide_classes`: (Optional) A comma-separated list of CSS classes to hide from the PDF.
- `--no_cache`: (Optional) Disable caching of PDF files.
